이벤트

1. 이벤트 모델의 종류

고전 이벤트 모델

특정 요소 객체를 가지고 와서 해당 요소의 이벤트 속성에 접근한 후
이벤트 핸들러를 연결하는 방식
이벤트를 제거할 수도 있다.(element.onclick = handler;)
동일한 이벤트 속성에 여러 개의 이벤트 핸들러를 연결할 수 없다 -> 한 개만 등록가능


인라인 이벤트 모델

요소 내부에 직접적으로 이벤트 속성을 제시해서 실행될 내용을 정의하는 방식
따로 script 코드에 작성된 함수를 호출하는 방식. html, js가 섞인다 -> 유지보수가 힘들다.

표준 이벤트 모델

w3c(world wide web consortium[웹 표준을 만드는 국제 기구])에서
공식적으로 지정한 이벤트 모델 element.addEventListener(이벤트명, 이벤트핸들러 함수);
동일 이벤트에 여러개의 핸들러를 등록할 수 있다.
-> removeEventListener()로 이벤트 제거 가능 (동일한 참조값을 넣어야한다.)
