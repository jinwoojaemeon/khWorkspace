정규표현식이란?

정규표현식(Regular Expression) 특정 문자열의 특정 패턴을 찾고, 검사하고, 치환하는 형식
정규표현식을 사용하면 문자열에 대한 특정 패턴 조건을 쉽게 검사할 수 있다.

정규표현식 객체 생성 및 관련된 메서드

메타문자

메타문자를 이용해서 정규식의 검색조건으로 삼을 '특정패턴' 제시 가능 문자열이 해당 정규식으로 제시한 특정 패턴에 만족하는지 test()통해 검사할 수 있다. ^ : 문자열의 시작 ex: /^j/.test("java") -> true
$ : 문자열의 끝 ex: /x$/.test("javascript") -> false
